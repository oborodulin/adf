; Список файлов, по которым не удаляются предыдущие версии.
; Этот файл понадобился, так как при поиске архивов под удаление предыдущих версий используется имя архива без версии.
; И так получается, что шаблон имени архива без версии приложения совпадает с шаблоном архива автономной справки: 
; например, erp*.ear и т.д. В этом случае, по шаблону, архив автономной справки (erp-help.ear) считается архивом приложения (erp.ear) и 
; удаляется наравне с ним. А так быть не должно, так как архив автономной справки пока не имеет версий.
; Для решения этой проблемы нужно либо переименовать архив справки таким образом, чтобы его шаблон не пересекался с
; архивом приложения, либо использовать этот файл
;==================================================
